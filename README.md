# AuraHair - Página Web E-commerce

## Propósito de la página

AuraHair es una página web dedicada a la venta de productos para el pelo.

El objetivo de la página es darle a los usuarios un espacio donde puedan conocer la marca, explorar diferentes productos, leer opiniones de clientes y comunicarse con la empresa mediante un formulario de contacto.


## Secciones de la página

### Inicio
La página principal presenta la marca AuraHair, sus objetivos y una introducción a sus productos destacados.

### Productos
Incluye un catálogo organizado con diferentes productos para el cabello, mostrando imágenes, descripciones y precios.

### Reseñas
Permite visualizar opiniones de clientes sobre los productos adquiridos, mostrando valoraciones y experiencias de uso.

### Contacto
Cuenta con un formulario para que los usuarios puedan comunicarse con la marca ingresando su nombre, correo electrónico y mensaje.
